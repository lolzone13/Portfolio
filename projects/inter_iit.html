About the project

The website was our submission for Inter IIT Tech Meet 10.0 High Prep Problem Statement Digital Alpha's SEC Filing Analyzer.It's a interactive dashboard through which investers can make wise decisions by looking at the charts and data.

The data is extracted from sec edgar database which was then analysed by sentiment analysis model and question answering models like Roberta. We provided the SEC filings through which we extracted the data and also highlighted it so that the user can easily go through it.


My Contribution to the project

  1. I was a part of the frontend team working with React.js
  2. I worked on Metrics table, charts generation and summary CSV Download functionality
