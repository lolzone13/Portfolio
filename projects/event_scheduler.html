Event Scheduler

Developed under Student's Web Committee IIT Guwahati

About the project

The main purpose behind this website was to solve the problem of mail spamming by different clubs within the campus. Clubs usually send mails to notify and remind people about their events. These mails usually get lost within a hoard of other mails and students miss out on good events

The portal is built with React for the frontend and Django Rest Framework for the Backend. It uses Microsoft OAuth for Authenticating users and Celery for sending emails. MapBox was used displaying locations within the campus. The website has many features like follow a club, meeting reminders etc.


My Contribution to the project

  1. I was involved in the developement of both the frontend and backend of the portal.
  2. In the backend I worked on developing the models and on authentication. I also worked on Django-Celery for sending electives.
  3. On the Frontend side I mainly developed the pages for the club side including pages like create event, update event, announcement and resource addition, reminders mails page etc
  4. Also tried working with sockets for realtime notifications
